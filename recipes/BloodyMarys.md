# Bloody Mary's

~~~ markdown-script
recipeHeader(objectNew( \
    'servings', 6 \
))
~~~

Mix the following in a pitcher:

~~~ markdown-script
recipeIngredients(arrayNew( \
    '2 C tomato juice', \
    '1 1/2 C vodka', \
    '1 C water', \
    '1/3 C lemon juice', \
    '1 tbsp soy sauce', \
    '1 tbsp Tabasco', \
    '1 tsp horseradish', \
    '1/4 tsp salt', \
    '1/4 tsp black pepper' \
))
~~~

Pour into ice-filled glasses with your favorite garnish and enjoy!
